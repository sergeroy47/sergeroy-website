# Serge Roy — Espace de mémoire, de création et de contemplation

Site web personnel de Serge Roy, éducateur, contemplatif et créateur, basé à Longueuil, Québec.

🌐 **Site en ligne :** [sergeroy47.github.io/sergeroy-website](https://sergeroy47.github.io/sergeroy-website)

---

## Contenu

- **À propos** — Présentation personnelle : trois jardins (la terre, l'intérieur, les idées)
- **Textes** — Écrits et réflexions sur la technologie, la nature, la contemplation et l'existence
- **Photos** — Galeries photographiques sur le jardin, la famille, les arbres et les roches
- **Poésie-Secours** — Collection de collages (photographie, texte manuscrit, fond peint) depuis les années 1980
- **Résonance** — Visualisation sonore aux fréquences contemplatives (396, 432, 528, 639 Hz)
- **Balados** — Enregistrements audio sur la pratique contemplative, le corps et la présence
- **Vidéos** — Créations audiovisuelles sur le jardin et la pratique contemplative
- **Contact** — Formulaire de contact

## Structure des fichiers

```
index.html          — Page principale
admin.html          — Interface d'administration du contenu
contenu.js          — Données du site (textes, photos, collages, vidéos, balados)
galeries.html       — Page des galeries photos
galerie-style.css   — Styles des galeries
galerie-moteur.js   — Moteur des galeries
collages/           — Images des collages Poésie-Secours
photos/             — Photographies
portraits/          — Portraits de Serge Roy
roches/             — Galerie de roches
textes/             — Textes en HTML
balados/            — Fichiers audio (.m4a)
videos/             — Fichiers vidéo et vignettes
```

## Technologie

Site statique en HTML, CSS et JavaScript pur — sans dépendances ni framework.
