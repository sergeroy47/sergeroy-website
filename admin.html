COMMENT UTILISER CE PANNEAU

💾 Sauvegarder — Enregistre vos modifications dans le navigateur. Vos changements apparaissent immédiatement sur le site.


⬇ Exporter contenu.js — Télécharge le fichier contenu.js mis à jour. Remplacez l'ancien fichier dans votre dossier de site par ce nouveau fichier. Utile si vous publiez le site en ligne.


Modifier dans TextEdit — Ouvrez le fichier contenu.js dans TextEdit. Faites vos modifications entre les guillemets. Sauvegardez. Rechargez le site dans Safari.


Ajouter une photo — Copiez votre fichier image (ex: jardin.jpg) dans le même dossier que index.html. Inscrivez le nom exact du fichier dans le champ correspondant ici.


Demander à Claude — Vous pouvez toujours envoyer un document Word à Claude et lui demander de l'ajouter au site automatiquement.
